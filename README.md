- Use process.py to process experiment data

## Example usage
- python main.py <poison mode> <detect mode> <H> <k>
- python Graph.py
- python process.py

- poison mode: target, data, model
- detect mode: monitor, norm, cosine, krum
- H K = 3.84 0.4, 3.28 0.5, 2.77 0.6
